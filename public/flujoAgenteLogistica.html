Logistics Assistance

End-to-end automation via WhatsApp

1
Client Request

"Hello, I'm looking for a loading slot to fulfill my contract 1234-5678"

2

Control Steps

  • ✓ Verifies that the sender is a client
  • ✓ Gets balance (tons) for the client's contract
  • ✓ Client and contract match: proceeds
  • ✓ Slot available, proceeds
  • ✓ If no slot available, informs the client
4
Confirmation Ready

"You just have been assigned a slot to fulfill your contract 1234-5678. Slot number: TRI-1234-10122015. Now, the remaining tons for contract 1234-5678 to fulfill is 90 tt."

3

Intelligent Assignment

The system automatically assigns the available slot

  • Queries slot table
  • Assigns unique slot from "available"
  • Updates database : "available" slot to "assigned"
  • Informs the admin/company that the slot has been assigned
